# Classical Image Analysis with scikit-image

This project explores **classical computer vision techniques** using Python's **scikit-image** library.  
Focuses on **image preprocessing, feature extraction, and segmentation** without deep learning.

## Skills & Tools
- Python, scikit-image, OpenCV, NumPy, Matplotlib  
- Techniques: filtering, edge detection, segmentation, feature extraction  

## Key Takeaways
- Understand classical CV pipelines  
- Visualize and extract features for analysis or ML tasks
