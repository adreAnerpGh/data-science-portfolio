# Emigration Patterns Analysis with Pandas

Analysis of historical emigration data from **Arstotzka** using **Pandas** and **Matplotlib**.  
Focuses on **time series trends, demographic patterns, and outlier detection**.

## Skills & Tools
- Python, Pandas, NumPy, Matplotlib, Seaborn  
- Techniques: resampling, moving averages, grouping, visualization  

## Key Takeaways
- Analyze trends by year, age, income, profession, and location  
- Identify anomalies and visualize emigration patterns clearly
